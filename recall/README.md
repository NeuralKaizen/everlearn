# recall

Retrieve context from your own **second brain** — answers from *your* notes, not generic
knowledge. Part of [loom](../README.md).

Ask "why do I use X?", "what did I learn about Y?", "what did I decide about Z?", or
"what's the state of project W?" — recall searches your vault (concepts / decisions /
logs / projects) and answers concisely, citing the source notes with `[[links]]`.

**Read-only:** it never writes or changes notes. If the vault has nothing on the topic,
it says so and offers to capture it.

## Triggers

- "¿por qué hago / uso / elegí X?"

- "¿qué aprendí / sé sobre Y?"

- "¿en qué está el proyecto W?"

To *learn* a new concept (not retrieve one), that's everlearn.

## Install

Ships with loom — see the [collection README](../README.md).
