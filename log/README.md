# log

Quickly capture a **learning, error, or reflection** the moment it happens — the fastest,
lightest capture in [loom](../README.md). One tiny note per entry, before the insight
evaporates.

Each note carries a `kind` (`learning | error | reflection`), gets tagged and linked into
the graph, and lands in your vault's `logs/` folder — so it resurfaces later through
recall and the graph.

## Triggers

- "aprendí que…", "TIL…", "me di cuenta…" → learning

- "se rompió por…", "perdí tiempo con…" → error

- "nota mental:…", "creo que…" → reflection

Not for deep concepts (that's everlearn) or decisions (that's decide).
