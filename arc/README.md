# arc

Maintain the living **hub** for a project — its context, lifecycle, and links to
everything captured under it. The slow, evolving layer of [loom](../README.md): few notes,
large, updated over time. A project's arc, from start to ship.

arc keeps one note per project in your vault's `projects/` folder, tracks its status
(`active | paused | shipped | abandoned`), and **aggregates** the decisions, logs, and
concepts that point at it via `project:` — turning scattered captures into a project's
story.

## Triggers

- **Create:** "arrancá / creá el proyecto X"

- **Update:** "actualizá el proyecto X", "X pasó a shipped / en pausa", "¿en qué está X?"

Not for a single decision/log/concept — those are decide / log / everlearn.
